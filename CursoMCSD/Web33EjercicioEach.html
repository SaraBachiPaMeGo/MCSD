Ejercicio con each

Realizar una aplicación dónde sumaremos el contenido de varias cajas de texto.

Las cajas estarán dibujadas dentro del código HTML.

Al pulsar sobre iniciar, generaremos números aleatorios sobre las cajas.

Al pulsar sobre Sumar, sumaremos el valor de todas las cajas.

Al perder el foco de cada caja, cambiaremos el número (value) de las cajas y le asignaremos un color random.
